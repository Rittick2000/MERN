There are Three Types of list---
1) Ordered List
2) Unordered List
3) Descreptive List

Unordered List (ul)

li= list item

  • Maggie
  • Kurkure
  • Lays

Ordered List (ol)

ol>li*3 then tab to create three list item in ordered list
